Традиционная подборка статей и материалов по теме Quarkus.
В этот раз она получилась больше чем обычно и содержит много материалов, которые могут быть полезны тем, кто только начинает знакомиться с Quarkus:
- Kubernetes Service Discovery and Selection with Stork - эта статья описывает как можно использовать SmallRye Stork в Kubernetes кластере. Stork - фреймворк для балансировки нагрузки на стороне клиента и обнаружения сервисов (service discovery);
- Quarkus Superheroes to the Rescue! - статья описывает пример создания микросервисного приложения при помощи Quarkus показывая как запускать его локально или в Kubernetes;
- Validate Kubernetes Deployment in CI/CD with Tekton and Datree - статья рассказывает о том как валилировать Kubernetes манифест-файлы при помощи инструментов Tekton и Datree. В качестве тестового приложения используется сервис, написанный на Quarkus;
- Quarkus 2.7.1 Released - Why Quarkus? - статья может быть полезна разработчикам, которые только пытаются понять что такое Quarkus и какие особенности вылеляют его среди других Java фреймворков, а также кратко описывает что нового включает в себя версия 2.7.1;
- Build a REST API from the ground up with Quarkus 2.0 - ещё один очень подробный пример создания простого REST сервиса при момощи Quarkus. Может быть очень полезна тем, кто собирался попробовать Quarkus, но всё не решался;
- REST API using Quarkus and Panache - похожая на предыдущую статья, с ещё одним примером создания простого REST сервиса;
- Is it gonna be your first API with Kotlin and Quarkus? Let’s see! — Part 1 - третья статья о создании REST API при помощи Quarkus, но уже на Kotlin;
- Cloud Native Java: Integrating YugabyteDB with Spring Boot, Quarkus, and Micronaut - статья описывает как интегрировать YugabyteDB с популярными Java фреймвокрами - Quarkus, Spring Boot и Micronaut;
- Building a Knowledge Base Service With Neo4j, Kafka, and the Outbox Pattern - пример создания сервиса для базы знаний с использованием Quarkus, Kotlin, Neo4j и Kafka.

Многие из перечисленных выше статей имеют ссылки на репозитории с исходным кодом, который можно склонировать локально и запускать у себя для более глубокого изучения.
